**dynaris** is a JAX-powered Python library for Dynamic Linear Models (DLMs).

[ROADMAP](ROADMAP.md)

## Installation

You can install the **dynaris** library using `uv`:

```bash
uv add dynaris
```

## Getting Started

```python
import dynaris
```
